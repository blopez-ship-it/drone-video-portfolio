# Drone Video Portfolio

A GitHub Pages-ready portfolio for showcasing drone video work.

## Add Your Videos

Put your MP4 files in `assets/videos` and use these filenames to start:

- `showreel.mp4`
- `real-estate-tour.mp4`
- `site-inspection.mp4`
- `event-recap.mp4`
- `landscape-cinematic.mp4`

You can also edit `index.html` and change the `<source src="...">` paths to match your real video filenames.

## Customize The Site

Open `index.html` and replace:

- `Drone Video Portfolio` with your name or business name
- `your.email@example.com` with your email address
- The service descriptions with the kind of drone work you want
- The video titles and descriptions with your real project names

Open `styles.css` if you want to change colors, spacing, or typography.

## Publish On GitHub Pages

1. Create a new repository on GitHub.
2. Upload all files and folders from this project.
3. In the GitHub repository, open **Settings**.
4. Open **Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and `/root`, then save.
7. GitHub will give you a public website link after it finishes deploying.

## Local Preview

You can open `index.html` directly in a browser. The layout will work, but the video players need actual MP4 files in `assets/videos` before they can play.

## Video Tips

- Keep your main reel under 90 seconds.
- Export web clips as MP4 using H.264 video and AAC audio.
- Compress large clips before uploading so the site loads quickly.
- Put your strongest work first.
- Use project titles that match the jobs you want: real estate, inspection, construction, commercial, events, or landscape.
